# قفل الفكرة الأساسية — بانكو منصة إعلانات

**Owner أمر:** كمّل الإصلاح **بدون هدم الفكرة الأساسية للإعلانات تماماً.**

---

## الحقيقة الثابتة

| مبدأ | معنى تشغيلي |
|------|-------------|
| **الإعلان أولاً** | اكتشاف الأصول / الأقسام / إنشاء الإعلان هو المنتج الأساسي |
| **التمويل تسليم** | بعد طلب تمويل على إعلان، الأدمن يدرس ويحوّل للمؤسسة — بانكو وسيط إعلانات لا مقرض |
| **Banks هب تشغيلي** | brochure + inbox للأعضاء المربوطين — **ليس** سوق شركاء بديل عن الإعلانات |
| **التوثيق ≠ فتح منتج جديد** | Verify شارة ثقة؛ الـ inbox يحتاج ربط `owner_user_id` / seat من الأدمن |

---

## ممنوع في أي موجة بلا Start صريح منفصل

1. دليل بنوك حي عام يستبدل أو ينافس Feed الإعلانات (OPEN-02 / D1=ج)  
2. جعل Discover/Stay بوابة تمويل بدل إعلانات  
3. Melt أقسام أو ENTER صفوف  
4. ادّعاء أن التوثيق وحده يفتح تمويل/inbox  
5. تصنيع شركاء / قوائم وهمية  

---

## مسموح جراحياً (يقوّي الحلقة دون هدم)

- AuthZ / state machine / صدق النسخ  
- مساعدة أدمن: Verify ثم ربط owner لوسيط موجود  
- كشف UUID للحساب لربط آمن  
- إصلاح شاشة توثيق FI حتى لا تُعامل كتاجر إعلانات فقط
