## A) اختبارات آلية (يجب أن تبقى خضراء)

| # | أمر / بوابة | من يملك | ثقة عند الأخضر |
|---|-------------|---------|----------------|
| T1 | GitHub CI على PR المعني (Typecheck · API · ESLint · GCP gate · Mobile static) | من يفتح PR | عالية — بيئة CI |
| T2 | `pnpm --filter @workspace/banco-mobile run test:section-guard` | بعد دمج/على فرع W1 | عالية لعقد الفصل |
| T3 | `node --test tests/icons.test.mjs` (من مجلد الموبايل بعد deps) | أي مساس أيقونات | عالية لأندرويد tofu |
| T4 | `pnpm --filter @workspace/api-server test` | أي مساس API/FI | عالية للمنطق الخلفي |
| T5 | `pnpm run typecheck` + lint | كل PR كود | عالية للانحدار |

**ملاحظة بيئة Cursor Cloud:** بدون `node_modules` كاملة قد يفشل icons محلياً (`MODULE_NOT_FOUND`) — **لا تُحسب انحداراً** إن CI Mobile على #32 SUCCESS. أعد بعد `pnpm install`.
