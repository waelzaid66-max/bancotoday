# 06 — موجات ممنهجة واحدة تلو الأخرى + اختبار القبول

**القاعدة:** لا تبدأ موجة N+1 قبل إغلاق اختبار قبول الموجة N وموافقتك المكتوبة.  
**هذا الملف خطة — التنفيذ لاحقاً وبعد اعتماد الشيت 02.**

---

## نظرة الموجات

```
W0 Ops رؤية الحقيقة
 → W1 وقاية الفصل + إثبات Discover
 → W2 استكمال FI (#28)

```

## ماذا يفعل الوكيل بعد كل موجة (صارم)

1. يعرض diff والنطاق فقط  
2. يشغّل اختبارات القبول المتفق عليها  
3. يحدّث صف الموجة  
4. **يتوقف** وينتظر موافقتك للموجة التالية  

لا «نكمل الباقي واحنا شغالين» بدون بوابتك.
