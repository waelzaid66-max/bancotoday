# Deployment Guide (RC)

## 1. Backend (Replit / any Node 24 host)
```bash
git pull
pnpm install
pnpm --filter @workspace/db run push-force        # creates all tables incl. reference_* + price_observations
pnpm --filter @workspace/api-server run seed        # base taxonomy (locations, cars)
pnpm --filter @workspace/api-server run seed:reference        # Egypt geo/real-estate reference
pnpm --filter @workspace/api-server run backfill:observations # real price history from existing listings
pnpm --filter @workspace/api-server run build && pnpm --filter @workspace/api-server start
```

### Required secrets (Replit Secrets — never in code)
| Secret | Purpose |
|---|---|
| `DATABASE_URL` | Postgres |
| `CLERK_SECRET_KEY` / `CLERK_PUBLISHABLE_KEY` | auth |
| `OPENAI_API_KEY` | AI assistant (model auto‑selects `gpt-4o-mini`) |
| `RESEND_API_KEY` | transactional email (else log‑only) |
| `PAYMOB_SECRET_KEY` / `PAYMOB_PUBLIC_KEY` / `PAYMOB_HMAC_SECRET` / `PAYMOB_INTEGRATION_IDS` | payments |
| Object Storage credentials | media upload |

Also in the **Clerk dashboard**: enable Email OTP, and Google/Apple OAuth with redirect URIs.

## 2. Web apps (admin-os / dealer-os / landing)
Build on Linux (CI already does this):
```bash
pnpm --filter @workspace/admin-os --filter @workspace/dealer-os --filter @workspace/landing run build
```
Serve the `dist/` of each behind the chosen domains.

## 3. Mobile (EAS)
- `app.json`: name **BANCO OOM**, version **1.0.0**, Android `versionCode` **1**, package/bundle **com.bancooom.app**, EAS project `45f092c8-…`, owner `waelzaid`.
- Permissions are generated by plugins: camera + photos (expo-image-picker), location (expo-location), notifications (expo-notifications).
- **Action before store submit:** `plugins → expo-router → origin` is currently `https://replit.com/`. Set it to the **production domain** (universal/app‑link origin) so deep links resolve to BANCO, not Replit.
- Build: `eas build --platform android` / `--platform ios`; set iOS `buildNumber` explicitly for resubmissions.

## 4. Release gate
Do NOT push/submit unless: `pnpm run typecheck` = 0, api tests green, CI build green. (Enforced this RC.)
